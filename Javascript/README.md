# Javascript Minimal API

### a. Source Code
- `index.js` (Entry point using Express)

### b. Package Manager
- **npm** (Node Package Manager)

### c. Dependency File
- `package.json` (Defines `express` and other dependencies)
- `package-lock.json` (Dependency lockfile)

### d. App Runtime & App Package / OS Executable
- **Runtime**: Node.js Engine (V8)
- **Package**: Run directly via the V8 interpreter.

To build and run:
```bash
npm install
node index.js
```
