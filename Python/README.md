# Python Minimal API

### a. Source Code
- `main.py` (Entry point using FastAPI)

### b. Package Manager
- **pip** (Python Package Installer)

### c. Dependency File
- `requirements.txt` (Defines `fastapi` and `uvicorn`)

### d. App Runtime & App Package / OS Executable
- **Runtime**: Python 3.x Interpreter
- **Package**: Uvicorn ASGI Server

To build and run:
```bash
pip install -r requirements.txt
python main.py
```
