# C# Minimal API

### a. Source Code
- `Program.cs` (Entry point using .NET 8 Minimal APIs)

### b. Package Manager
- **NuGet** (Standard .NET package manager)

### c. Dependency File
- `swp_project_cs.csproj` (XML project file tracking NuGet packages and SDKs)

### d. App Runtime & App Package / OS Executable
- **Runtime**: .NET Core / .NET 8 CLR
- **Executable**: `dotnet publish` creates a portable `swp_project_cs.exe` or `.dll` package.

To build and run:
```bash
dotnet run
```
